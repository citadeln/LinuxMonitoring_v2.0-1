## Установить и настроить Prometheus и Grafana на виртуальную машину

`Prometheus` — это база данных временных рядов (TSDB). Его главная задача — собирать метрики (числовые данные во времени), хранить их и предоставлять язык запросов для извлечения этих данных. Он хранит историю.
Пример вопроса: Какое среднее значение загрузки процессора было за последние 5 минут?
```
sudo apt-get install -y prometheus stress
sudo systemctl enable prometheus
sudo systemctl status prometheus --no-pager
```

`Grafana` — это инструмент визуализации. Его главная задача — красиво рисовать графики на основе данных, которые он получает из различных источников (включая Prometheus). Он ничего не хранит сам по себе, он лишь отображает то, что лежит в базе данных.
Пример вопроса: Покажи мне график загрузки процессора в реальном времени на красивом экране.
```
sudo wget https://dl.grafana.com/oss/release/grafana_10.4.18_amd64.deb
sudo dpkg -i grafana_10.4.18_amd64.deb
sudo apt-get install -f # поиск и установка недостающих пакетов
sudo systemctl enable grafana-server # Включит автозагрузку (чтобы Grafana запускалась сама после перезагрузки сервера)
```

## Получить доступ к веб интерфейсам Prometheus и Grafana с локальной машины

- Настроить порты на виртуальной машине.
- Настроить сетевое соединение между вашим компьютером и ВМ.

Для Grafana: http://127.0.0.1:3000
```
Имя: grafana
Протокол: TCP
Адрес хоста: (оставьте пустым)
Порт хоста: 3000
Адрес гостя: (оставьте пустым)
Порт гостя: 3000
Для Prometheus (порт 9090)
```

Для Prometheus: http://127.0.0.1:9090
```
Имя: prometheus
Протокол: TCP
Адрес хоста: (оставьте пустым)
Порт хоста: 9090
Адрес гостя: (оставьте пустым)
Порт гостя: 9090
```
